This project focuses on **Bio-Computing** by applying deep learning techniques to medical imaging.

### Key Features
* **Architecture**: Custom CNN designed for high-resolution cell images.
* **Accuracy**: Achieved high accuracy on the test set using standard benchmarks.
* **Dataset**: Utilized the NIH Malaria Dataset.

### Technical Details
We implemented the model using **PyTorch** and utilized **transfer learning** to optimize training time.
